# Online Bookstore Website

This project is a full-stack Online Bookstore web application that allows users to browse books, add them to a shopping cart, and complete purchases.

## Technologies Used
- HTML, CSS, JavaScript
- Node.js
- Express.js
- MongoDB (Mongoose)

## Features
- View featured books
- View book details
- Add and remove books from cart
- Checkout and order placement
- Automatic hiding of purchased books

## Backend Setup Instructions
1. Download `backend.zip`
2. Extract the files
3. Run `npm install`
4. Create a `.env` file and add MongoDB connection string
5. Run `node server.js`

## Note
The `node_modules` folder is excluded from the repository.
Dependencies can be installed using `npm install`.
